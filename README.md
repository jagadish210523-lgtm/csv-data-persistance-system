# Student Records Manager

A command-line interface (CLI) application built in Python designed to handle student enrollment workflows. This project demonstrates Object-Oriented Programming (OOP) design principles and file-based data persistence.

## 🚀 Key Features
* **Data Persistence:** Automatically initializes and writes data to a localized CSV database.
* **Full CRUD Operations:** Supports creating, reading, searching, and deleting user records.
* **State Management:** Utilizes active runtime memory (`student_db` list) synchronized with flat-file storage.
* **Input-Driven Workflows:** Employs an interactive user menu loop with dynamic data routing.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Storage Platform:** Flat-file CSV
* **Core Modules:** `os` (File-path verification)

## 📦 Quick Start
1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate to the project folder:
   ```bash
   cd student-records-manager
   ```
3. Run the application:
   ```bash
   python main.py
   ```
